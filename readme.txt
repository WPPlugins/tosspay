== Description ==

This plugin will work with the Toss App payment.
Toss App is based on a bank account transfer.

Supported Features.
*   Payment with TossPay - Toss App.
*   Automatic Refund.
*   Work with the TossPay Admin page.
*   Callback from TossPay Server. It will change order status.

== Installation ==

1. Upload 'TossPay' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go Admin page - Woo Commerce - Settings - checkout - Select 'TossPay' Tab.
4. Activate TossPay.
5. Go http://toss.im/tosspay for get the API Key.
6. Insert API Key and Save Settings.
7. Then it works!
